要件定義書

概要

この文書は docs-portal における Markdown 閲覧・編集・差分確認の要件をまとめます。

画面要件

  • ブラウザ上で Markdown を編集できる
  • HTML プレビューを即時確認できる
  • 保存前に前版との差分を確認できる
  • 非エンジニアにも分かる表示にする

旧仕様では Markdown ファイルをダウンロードして編集していました。
新仕様ではブラウザ上で編集し、プレビューと差分を確認してから保存します。
