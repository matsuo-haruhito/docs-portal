Markdown preview / diff viewer design reference

この mock は実画面そのものではありません。版詳細画面の current support は runbook を正本にし、この静的資料では後続 issue で扱う proposal と並べて境界を読み分けます。Current support が増えた場合は、この note と下の Current support / Proposal only の区分を同じ更新で見直します。

Current support

  • 版詳細を read-only の確認ハブとして表示する
  • HTML本文、Markdown行差分、HTML差分、表セル差分へ移動する
  • 添付・元ファイル、品質チェック、比較対象版を確認する
  • 検索補助、codeblock copy / JSON検証、Markdown table toolbar は各 runbook の current support に従って読む
  • 押せる導線は実在する viewer / 版詳細内の移動先として扱う

Proposal only

  • ブラウザ上での Markdown 編集保存
  • server-side dry-run や codeblock action の実行・監査 workflow
  • Markdown table toolbar の full preference integration
  • 保存やコメント追加の workflow 変更
